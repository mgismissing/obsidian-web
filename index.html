So, what is Obsidian exactly?

Obsidian is an isolated group of webpages that are completely different from the World Wide Web.

For example html doesn't exist and is replaced by Shard for layout and Ruler for styles. The only supported server side language is Frame, but you can also use php and the other traditional ones (but it's recommended you don't because even the file naming is different, see File Naming).

How to access Obsidian websites

Obsidian is a closed network, so only a selected group of people can see Obsidian websites. And even if you found the websites in a normal browser, how would you read them? Remember, no HTML/CSS in Obsidian. There is only Shard and Ruler.
